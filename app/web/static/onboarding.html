👁 Заведи своего Большого Брата

Бот читает твои переписки в Telegram и сам раскладывает договорённости по TickTick. Разворачивается ПОЛНОСТЬЮ у тебя: свой бот, своя база, свой TickTick — автор оригинала ничего не видит.

1 Разверни свой экземпляр

Выбери один из способов ниже. Оба разворачивают одно и то же — просто разные пути.

🚀 Развернуть на Railway (один клик) 📦 Репозиторий и инструкция (DEPLOY.md)
Или — команда для терминала

Вставь в Терминал (Mac: Cmd+Space → Terminal; Windows: WSL или Git Bash). Подставь свои значения вместо плейсхолдеров — токен бота из @BotFather и свой Anthropic API ключ.

Скрипт сам поставит Railway CLI, залогинит и всё развернёт — дальше следуй подсказкам в терминале.

Опционально — свой n8n (автоматизация)

Отдельный сервис для дополнительных автоматизаций (например почтовый пайплайн). Не обязателен для самого бота.

2 Настрой Telegram и TickTick

В @BotFather у своего бота включи Secretary Mode (бывший Business Mode) и выключи Group Privacy.

TickTick — отдельный маленький сервер (ticktick-mcp), который тоже разворачивается у тебя. Когда получишь его адрес вида https://…/mcp/<секрет> — напиши своему боту в личку:

/connect https://<твой-ticktick-mcp>.up.railway.app/mcp/<секрет>
